bool isPrime ( uint64_t  num)

Determines whether or not a given number is a prime number.

This function determines whether or not the given number is a prime number. If the number is 2 or 3, ir will return true. It will rule out numbers less than 2, divisible by 2, and divisible by 3. It will then loop up to the square root of n checking for divisibility. If none is, the number is a prime.

Parameters
numThe number to be tested for primality.
Returns
True if number is a prime, false if not.
